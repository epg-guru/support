# epg.guru support

Issue tracker for [epg.guru](https://epg.guru/), an XMLTV EPG (electronic program guide) site.

## Reporting an issue

Open a [new issue](../../issues/new/choose) and pick the template that fits:

- **Data Issue**: guide data is wrong or missing, incorrect episode numbers, missing images, wrong titles/descriptions, wrong air times, etc.
- **XMLTV Parsing Error**: the downloadable XML file itself is malformed or fails to parse.
- **General Question / Other**: anything else.

> Guide data comes from our upstream data provider. We can't fix upstream data errors directly, and we can't add channels, markets, or data that aren't supported. Data issue reports are tracked, but fixes aren't guaranteed.
